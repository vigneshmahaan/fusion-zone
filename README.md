# Fusion Shoe Zone & Accessories

A premium, modern web application for discovering premium shoes, stylish men's accessories, and kids' fashion. Built with React, TanStack Start, and Tailwind CSS.

## Features

- **Shop by Category:** Easily browse through curated collections including Shoes, Men's Accessories, and Kids' Fashion.
- **Trending & New Arrivals:** Stay updated with the latest fashion drops and best-selling items.
- **Interactive Experience:** Smooth micro-animations, premium gradients, and interactive layouts.
- **Dynamic SEO:** Fully search engine optimized with specific meta tags for all main pages.

## Technologies Used

- **Framework:** React + TanStack Start (for modern SSR/Hydration)
- **Styling:** Tailwind CSS + Vanilla CSS variables for premium look & feel
- **Icons:** Lucide React
- **Animations:** Framer Motion

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Development

Install the dependencies:

```bash
npm install
# or
bun install
```

Start the development server:

```bash
npm run dev
# or
bun run dev
```

### Production Build

To build the application for production:

```bash
npm run build
# or
bun run build
```

## Vercel Deployment

This project is configured to deploy seamlessly to Vercel using the Nitro bundler.
Simply connect the repository to your Vercel account, and it will be auto-detected and built.
